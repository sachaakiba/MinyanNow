# MinyanNow

**Trouvez votre minyan. Rejoignez votre communaute. En un instant.**

---

## Le probleme

Chaque jour, des milliers de Juifs cherchent un minyan pour prier. Que ce soit pour une Mincha en semaine, des Sheva Berakhot de derniere minute, ou une Brit Mila dans un quartier que l'on ne connait pas, la meme question revient :

> *"Ou trouver 10 hommes, maintenant, pres de chez moi ?"*

Les groupes WhatsApp debordent, les appels se multiplient, et souvent le minyan tombe a l'eau faute d'organisation. Le temps presse, les gens sont disperses, et personne n'a une vision claire de qui est disponible, ou, et quand.

---

## La solution : MinyanNow

MinyanNow est une application mobile qui connecte en temps reel les membres de la communaute juive autour d'evenements religieux. Plus besoin de chercher : l'evenement vient a vous.

### Comment ca marche

1. **Ouvrez l'application** -- la carte affiche instantanement les evenements a proximite
2. **Touchez un evenement** -- consultez les details, le nombre de places, le type de priere
3. **Demandez a participer** -- l'organisateur valide votre identite et vous accepte
4. **Lancez le GPS** -- Waze ou Google Maps vous guide directement sur place

C'est aussi simple que ca.

---

## Fonctionnalites

### Carte interactive

- Visualisez tous les evenements proches sur une carte en temps reel
- Les evenements sont regroupes intelligemment par zone (clustering)
- Basculez entre vue carte et vue liste selon votre preference
- Les evenements urgents (moins de 3 places restantes) sont mis en evidence

### Types d'evenements

| Type | Description |
|------|-------------|
| **Mincha** | Priere de l'apres-midi |
| **Arvit** | Priere du soir |
| **Sheva Berakhot** | Benedictions des sept jours apres un mariage |
| **Brit Mila** | Circoncision |
| **Autre** | Tout autre evenement necessitant un minyan |

### Creation d'evenement

- Creez un evenement en quelques secondes : type, date, heure, adresse
- L'adresse est completee automatiquement (Google Places)
- Definissez le nombre maximum de participants
- Ajoutez des participants initiaux deja presents (famille, amis)
- Votre evenement apparait immediatement sur la carte pour tous les utilisateurs proches

### Systeme de participation securise

MinyanNow ne se contente pas d'un simple "je participe". L'application met en place un systeme de confiance :

1. **Demande de participation** -- l'utilisateur envoie une demande a l'organisateur
2. **Verification d'identite** -- l'organisateur peut consulter les documents du demandeur (piece d'identite)
3. **Validation** -- l'organisateur accepte ou refuse la demande
4. **Notification instantanee** -- le participant est notifie de la decision

Ce processus garantit que seuls des membres verifies de la communaute participent aux evenements.

### Verification des documents

Chaque utilisateur fournit trois documents lors de son inscription :

- **Piece d'identite** -- Pour verifier l'identite reelle
- **Ketouba** -- Certificat de mariage, preuve d'appartenance a la communaute
- **Selfie** -- Pour confirmer que le proprietaire du compte est bien la personne sur les documents

Les documents sont :
- Chiffres et stockes de maniere securisee
- Verifies par une equipe d'administration
- Consultables uniquement par l'organisateur lors d'une demande de participation

### Notifications intelligentes

Ne ratez plus jamais un minyan :

- **Alerte de proximite** -- Un evenement se deroule pres de vous ? Vous etes automatiquement prevenu, meme si vous ne cherchiez pas
- **Nouvelles demandes** -- En tant qu'organisateur, soyez alerte des qu'une personne souhaite rejoindre votre evenement
- **Statut de participation** -- Sachez immediatement si votre demande est acceptee ou refusee
- **Rappel avant l'evenement** -- Un rappel ~1h avant le debut pour ne pas oublier
- **Badge sur l'icone** -- Un indicateur visuel dans la barre de navigation vous montre le nombre de demandes en attente

Le rayon de detection de proximite est configurable de 100 m a 2 km selon vos preferences.

### Navigation GPS

Depuis la fiche d'un evenement, lancez directement :
- **Waze** -- Navigation guidee en temps reel
- **Google Maps** -- Itineraire vers l'adresse de l'evenement

Un seul tap et vous etes en route.

### Profil et parametres

- Renseignez votre prenom, nom, nom hebreu, synagogue d'appartenance
- Consultez le statut de verification de vos documents
- Changez la langue de l'application a tout moment
- Gerez finement vos preferences de notification

## Securite et confiance

La confiance est au coeur de MinyanNow :

- **Authentification par telephone** -- Connexion par code OTP envoye par SMS, pas de mot de passe a retenir
- **Triple verification documentaire** -- Piece d'identite, Ketouba et selfie valides par une equipe d'administration
- **Controle par l'organisateur** -- Chaque organisateur decide qui participe a son evenement apres verification
- **Donnees chiffrees** -- Les documents sensibles sont stockes de maniere securisee sur le cloud
- **Conformite RGPD** -- Politique de confidentialite et conditions d'utilisation claires et transparentes

## L'experience en images

### Ecran d'accueil
Ouvrez l'application et voyez instantanement les evenements autour de vous. Les points sur la carte s'animent, les clusters indiquent les zones actives. Un evenement urgent ? Il brille en orange.

### Fiche evenement
Toutes les informations essentielles : type de priere, date, heure, nombre de places, organisateur. Un bouton pour rejoindre, un autre pour naviguer. Simple et efficace.

### Gestion des demandes
En tant qu'organisateur, consultez les demandes en attente, verifiez l'identite des participants, acceptez ou refusez en un geste. Le badge rouge dans la barre de navigation vous previent qu'une action vous attend.

### Alertes de proximite
Vous marchez dans la rue et votre telephone vibre : un minyan demarre a 200 metres dans 30 minutes. Vous n'aviez rien prevu, mais le minyan est la. Rejoignez-le.

## Architecture technique

Pour les curieux et les developpeurs :

| Composant | Technologie |
|-----------|-------------|
| Application mobile | React Native + Expo SDK 54 |
| Langage | TypeScript (strict) |
| Navigation | React Navigation v7 |
| Authentification | Better Auth (OTP SMS via Twilio) |
| Base de donnees | PostgreSQL (Neon) + Prisma ORM |
| Serveur API | Express.js |
| Stockage documents | Cloudinary (chiffre) |
| Notifications push | Expo Notifications |
| Cartes | Google Maps (react-native-maps) |
| Internationalisation | i18next |
